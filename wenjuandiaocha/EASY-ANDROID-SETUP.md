# 简单在Android设备上使用云护童心应用

由于直接下载安装文件遇到问题，我们可以使用一种更简单的方法在Android设备上使用应用，无需安装任何额外软件。

## 方法：通过本地服务器访问

### 步骤1：启动本地服务器

1. **双击运行 `start-server.bat` 文件**：
   - 这会打开一个命令提示符窗口
   - 您会看到 "正在启动本地服务器..." 的消息
   - 服务器会在端口 8082 上运行

2. **获取您电脑的IP地址**：
   - 打开一个新的命令提示符窗口
   - 输入 `ipconfig` 命令
   - 找到 "IPv4 地址" 一行，记录下您的IP地址（例如：192.168.1.100）

### 步骤2：在Android设备上访问应用

1. **确保Android设备和电脑连接在同一网络**：
   - 两者都连接到同一个Wi-Fi网络

2. **在Android设备上打开Chrome浏览器**

3. **在地址栏中输入**：
   ```
   http://您的IP地址:8082
   ```
   例如：`http://192.168.1.100:8082`

4. **添加到主屏幕（可选）**：
   - 点击Chrome浏览器右上角的菜单按钮（三个点）
   - 选择 "添加到主屏幕"
   - 点击 "添加" 按钮
   - 现在您可以从主屏幕直接启动应用

### 步骤3：使用应用

1. **选择角色**：点击 "学生端"、"教师端" 或 "管理员端"
2. **填写测评问卷**：按照提示完成心理健康测评
3. **获取分析结果**：提交后，系统会进行AI分析并显示结果
4. **查看资源包**：根据分析结果，您可以查看推荐的资源包

## 优势

1. **无需安装任何软件**：利用现有的Chrome浏览器
2. **操作简单**：只需几个步骤即可完成设置
3. **更新方便**：只需更新电脑上的文件，Android设备上的应用会自动更新
4. **跨平台**：同样的方法也适用于iOS设备、平板电脑等

## 注意事项

1. **网络连接**：Android设备和电脑需要在同一网络中
2. **服务器运行**：在使用应用时，电脑上的服务器窗口需要保持打开状态
3. **浏览器**：建议使用Chrome浏览器，因为它对Web App的支持最好
4. **权限**：应用需要网络访问权限才能使用AI分析功能

## 故障排除

### 问题：无法访问应用
**解决方案**：
- 检查电脑和Android设备是否在同一网络
- 检查服务器窗口是否显示 "Serving HTTP on 127.0.0.1 port 8082" 的消息
- 尝试关闭防火墙或添加例外规则

### 问题：应用加载缓慢
**解决方案**：
- 确保您的网络连接稳定
- 尝试刷新页面

### 问题：AI分析失败
**解决方案**：
- 检查网络连接是否正常
- 确保您的网络可以访问Aliyun API服务

## 停止服务器

当您完成使用应用后：

1. **回到服务器窗口**
2. **按下 Ctrl+C 组合键**
3. **按任意键关闭窗口**

## 总结

通过这种方法，您可以在不需要安装任何额外软件的情况下，在Android设备上使用云护童心应用。这种方法简单、快速，并且对于大多数用户来说已经足够使用。
